# 周报 / 月报

追踪 AI Agent 领域的最新动态，持续更新。

## 2026 年 5 月月报

### 行业动态

**Anthropic**
- Claude Opus 4.7 和 Sonnet 4.6 发布，Agent 能力大幅提升
- Claude Code 持续迭代，支持 MCP 扩展
- MCP 协议生态快速扩展，社区 Server 数量超过 100 个

**OpenAI**
- GPT-4o 多模态能力增强
- OpenAI Agents SDK 持续更新
- Codex CLI 编程 Agent 发布

**Google**
- A2A 协议发布，推动 Agent 间通信标准化
- Gemini 2.5 Pro 发布，长上下文能力增强

**国产模型**
- DeepSeek-R1 推理模型引起广泛关注
- GLM 系列持续迭代，工具调用能力提升
- Qwen2.5 系列发布，开源模型能力逼近闭源

### 技术趋势

1. **Agent 框架收敛** — LangGraph 逐渐成为主流选择，LangChain 简化为底层组件
2. **MCP 生态爆发** — 越来越多工具支持 MCP 协议，即插即用
3. **小模型 Agent** — Qwen2.5-7B、DeepSeek 等小模型做 Agent 越来越可行
4. **Agentic RAG** — 从静态检索升级为 Agent 自主检索，成为新标配
5. **Multi-Agent 落地** — 从概念验证走向生产部署
