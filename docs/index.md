## 关于

这是一个技术学习笔记项目，系统记录和分享 **AI 智能体（Agent）研发**过程中的学习资料与技术总结。

内容从基础概念到实战项目，循序渐进，适合想系统学习 AI Agent 开发的开发者。

- **[技术栈总览](/ai-agent/tech-stack)** — Agent 框架、向量数据库、开发工具一网打尽
- **[Prompt Engineering](/ai-agent/prompt-engineering)** — 提示词工程的系统化方法论
- **[Function Calling](/ai-agent/function-calling)** — 工具调用机制与实战详解
- **[RAG 技术](/ai-agent/rag)** — 检索增强生成的完整链路
- **[Multi-Agent](/ai-agent/multi-agent)** — 多智能体协作的模式与实践
- **[实战项目](/ai-agent/practice)** — 从零开始的动手项目指南
- **[学习资源](/ai-agent/resources)** — 精选文章、视频、开源项目
